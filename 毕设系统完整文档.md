# 新闻情感分析系统 - 毕业设计文档

## 一、系统概述

本系统是一个基于 **Django + PyTorch** 的中文新闻情感分析平台，实现了从新闻数据采集、预处理、情感分析到可视化展示的完整流程。

### 3.4 深度学习模型 (`model/`)

#### 3.4.1 模型架构 (`bilstm_attention.py`)

```
输入序列 → 词嵌入层 → Bi-LSTM → Attention → 全连接层 → 输出Logits
    ↓             ↓          ↓           ↓
  Token ID     128维       2×256维      加权池化    3分类
```

**模型参数**：

| 参数 | 值 | 说明 |
|------|-----|------|
| vocab_size | 动态 | 基于训练集构建 |
| embedding_dim | 128 | 词嵌入维度 |
| hidden_dim | 256 | LSTM隐藏层维度 |
| n_layers | 2 | LSTM层数 |
| output_dim | 3 | 三分类输出 |
| dropout | 0.5 | Dropout比例 |

#### 3.4.2 Attention机制

```python
class AttentionLayer(nn.Module):
    def forward(self, lstm_output):
        # 计算注意力打分
        score = torch.tanh(self.W(lstm_output))
        attention_weights = torch.softmax(self.u(score), dim=1)
        
        # 加权求和
        weighted_output = lstm_output * attention_weights
        context_vector = torch.sum(weighted_output, dim=1)
        
        return context_vector, attention_weights
```

**设计特点**：
- 对padding位置进行mask处理
- 线性注意力打分，增强可解释性
- 支持注意力权重可视化

#### 3.4.3 训练配置 (`train.py`)

**训练参数**：

| 参数 | 值 |
|------|-----|
| batch_size | 64 |
| learning_rate | 1e-3 |
| epochs | 20 |
| patience | 5 (早停) |
| optimizer | Adam |
| loss | CrossEntropyLoss |

**早停策略**：验证集F1分数连续5轮未提升则停止训练

## 九、系统特色

### 9.1 技术亮点

1. **Bi-LSTM + Attention 模型**：结合双向LSTM捕获上下文语义和注意力机制聚焦关键词汇
2. **自动化数据管道**：一键完成爬虫→预处理→预测→分析的完整流程
3. **实时情感分析**：支持单文本在线预测API
4. **可视化仪表盘**：丰富的图表展示情感分布和趋势
5. **用户认证系统**：完整的用户注册、登录、个人中心功能

### 9.2 数据处理特色

1. **多源数据采集**：支持新浪多个新闻板块
2. **智能文本清洗**：自动去除广告、HTML标签、特殊符号
3. **重复检测**：基于URL和内容哈希的双重去重
4. **繁简体转换**：支持中文繁简体自动转换
5. **停用词过滤**：基于自定义停用词表的词汇过滤

### 9.3 模型优化策略

1. **早停机制**：防止过拟合，节省训练时间
2. **学习率调度**：基于验证集损失自适应调整学习率
3. **梯度裁剪**：防止梯度爆炸
4. **混合精度训练**：支持GPU加速
5. **模型可解释性**：注意力权重可视化
